Featured Projects

Freelance Client - Elite Compass

This is a website that I created for one of my first freelance clients as a web developer. It features images and content provided by the client, a form, payment options, and it is mobile friendly.

Passion Project - Accommodations Assistant

Everyone needs a passion project! This is an app designed for teacher that uses a custom REST API to provide teachers with information about learning disabilities and ways they can accommodate for students in their classrooms.

Musician's Friend - Scale & Mode Generator

Building Projects that help others is my ultimate goal. When my husband asked if I could build a website to make practicing scales and modes a bit easier, it was a no-brainer. It features audio recordings of all of the scales and mode, recorded by my husband.

Fairy's Garden - Single-Page

Before I started freelancing, I started building websites that I thought would be useful to small businesses. Fairy's Garden was one of the first. It feature a contact form, outisde links to information about the products, and a grid all in a single-page design.
